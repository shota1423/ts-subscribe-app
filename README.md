# ts-subscribe-app

## 環境構築

### 環境

- Ubuntu 22.04.03 もしくは macOS
- Node.js v20.15.0
- Next.js 14.2.3
- TypeScript 5.4.5

### インストールするもの

- nvm

### 起動方法

1. リポジトリをクローンする

   ```bash
   git clone https://github.com/shota1423/ts-subscribe-app.git
   ```

1. アプリを起動する

   ```
   docker compose up --build
   ```
